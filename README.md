# numconv

## Description

numconv is a small program that converts any number from any numerical system to any number from any numerical system.

Currently only the numstoany.c and numstoany.h are working, feel free to use it in your own project.

Any ideas to improve are always appreciated.

## Usage

```
make
```
Then a output folder is generated
```
output/main <input numerical system> <number> <output numerical system>
```
